# DataView

Projeto de análise exploratória de dados utilizando Python, Pandas, NumPy, Matplotlib e Seaborn.

## Objetivo

Analisar dados de vendas, realizar limpeza, tratamento de outliers, geração de métricas, segmentação de clientes e visualizações gráficas.
